== [#int]#INT#

A function truncating a real number to the lower integer.

[source,console]
----
X=INT(Y)

INT(99.8)   =99
INT(-12)    =-12
INT(-12.1)  =-13
----

This function converts a real number (one with a decimal part) to the nearest integer (whole number) less than the number supplied. Thus,

[source,console]
----
INT(14.56)
----

gives 14, whereas

[source,console]
----
INT(-14.5)
----

gives -15.

=== Syntax

[source,console]
----
<n-var>=INT<numeric>
----

=== Associated Keywords

[source,console]
----
None
----
